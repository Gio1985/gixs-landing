# GIXS — Landing Page

> *Engineering the Invisible.*

Sitio web de "Stealth Launch" para **GIXS** ([gixs.dev](https://gixs.dev)). Diseñado con estética Apple-esque ultra-minimalista, micro-interacciones y rendimiento extremo.

## Stack Tecnológico

- **HTML5** semántico con SEO optimizado
- **CSS3** puro — variables, grid, animations, glassmorphism
- **JavaScript** vanilla — zero dependencias
- **Desplegable como Static Site** en Render, Netlify, Vercel o cualquier CDN

## Estructura

```
home_page/
├── index.html      # Página principal
├── styles.css      # Sistema de diseño completo
├── main.js         # Interacciones y animaciones
├── logo.png        # Ícono/logo GIXS
├── hero-bg.png     # Background atmosférico (opcional)
└── README.md       # Este archivo
```

2. Conectar este repositorio
3. Configurar:

   - **Publish Directory**: `./`
